# BPMN Monorepo

Monorepo inicial com frontend em React (Vite + TypeScript) e placeholder para backend.

## Estrutura

- `frontend/` aplicações e pacotes compartilhados em workspace PNPM.
- `backend/` placeholder para implementação futura.

## Comandos principais

```bash
pnpm -w install
pnpm -w dev
pnpm -w lint
pnpm -w format
```
